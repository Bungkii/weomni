คำถามที่พบบ่อย (FAQ)

รวบรวมทุกข้อสงสัยเกี่ยวกับระบบ FlexiPOS เพื่อให้คุณมั่นใจก่อนตัดสินใจใช้งาน

ระบบและการทำงาน

FlexiPOS ทำงานบนสถาปัตยกรรม Cloud-based เป็นหลัก เพื่อให้ข้อมูลเชื่อมต่อแบบ Real-time อย่างไรก็ตาม ระบบของเรามีกลไก Local Cache ที่ทำงานผ่านเบราว์เซอร์ ช่วยให้แคชเชียร์ยังสามารถคิดเงินและพิมพ์ใบเสร็จชั่วคราวได้เมื่ออินเทอร์เน็ตสะดุด และระบบจะซิงค์ข้อมูลกลับสู่ฐานข้อมูล Supabase อัตโนมัติทันทีที่การเชื่อมต่ออินเทอร์เน็ตกลับมาเป็นปกติ
ข้อมูลยอดขาย สต็อก และฐานข้อมูลลูกค้าของคุณทั้งหมด ถูกจัดเก็บไว้อย่างปลอดภัยบนระบบ Cloud Database ของ Supabase (PostgreSQL) ซึ่งตั้งอยู่บนศูนย์ข้อมูลมาตรฐานสากล (Sydney Region) คุณมีสิทธิ์ความเป็นเจ้าของข้อมูลร้านค้าของคุณ 100% และข้อมูลจะไม่มีการถูกนำไปส่งต่อให้แพลตฟอร์มอื่น
เราใช้ Payload CMS สำหรับระบบจัดการหลังบ้าน คุณสามารถเพิ่ม/ลดเมนูสินค้า จัดการหมวดหมู่ กำหนดโปรโมชัน ดูรายงานยอดขายแบบ Real-time และกำหนดสิทธิ์การเข้าถึงของพนักงาน (Admin / Cashier) ได้จากหน้า Dashboard เดียว

ฮาร์ดแวร์และอุปกรณ์

รองรับครับ! เนื่องจาก FlexiPOS พัฒนาเป็น Web-based Application (Next.js) หากคุณมีคอมพิวเตอร์ระบบ Windows, แท็บเล็ต หรือเครื่อง POS เดิมอยู่แล้ว สามารถเปิดใช้งานผ่านเบราว์เซอร์ได้ทันที ส่วนเครื่องพิมพ์ใบเสร็จและลิ้นชักเก็บเงิน สามารถเชื่อมต่อผ่านพอร์ต USB มาตรฐานได้เลย
สำหรับแพ็กเกจที่รองรับระบบ 2 หน้าจอ (เช่น Set C ที่ใช้จอ iMin Swan 2) หน้าจอที่สองจะหันเข้าหาลูกค้า โดยลูกค้าสามารถเห็นรายการสินค้า สแกน QR Code เพื่อชำระเงิน และสามารถกดเบอร์โทรศัพท์เพื่อใช้สิทธิ์สมาชิกหรือสะสมแต้มผ่านหน้าจอสัมผัสได้ด้วยตัวเอง ช่วยเพิ่มความรวดเร็วและความถูกต้องหน้าเคาน์เตอร์

ราคาและแพ็กเกจ

ค่าบริการรายเดือนและรายปีของเรารองรับการเชื่อมต่อร้านค้า 1 สาขา หากในสาขาเดียวกันต้องการเพิ่มเครื่องคิดเงิน คุณสามารถเปิดใช้งานบนเครื่องอื่นได้โดยไม่มีค่าใช้จ่ายซอฟต์แวร์เพิ่มเติม เพียงแค่คุณจัดเตรียมหรือสั่งซื้อฮาร์ดแวร์ (คอมพิวเตอร์/เครื่องปริ้น) เข้ามาเพิ่มในระบบเท่านั้น
ได้แน่นอนครับ FlexiPOS สร้างขึ้นเพื่อให้ยืดหยุ่นที่สุด เรามีการจัดเลย์เอาต์ใบเสร็จแบบคลีน (ตัดคำว่า Reprint ออก ใช้ฟอนต์ IBM Plex Sans Thai/Sarabun) ให้เป็นมาตรฐาน แต่หากองค์กรของคุณต้องการเพิ่มฟอร์มเฉพาะ หรือฟีเจอร์พิเศษ (Customization) สามารถแจ้งทีมงานเพื่อประเมินค่าบริการเพิ่มเติมได้
เรามีทีม Support ดูแลคุณตลอด 24/7 ผ่านช่องทาง LINE Official (@FlexiPOS) และ Call Center พร้อมระบบ Remote Support เพื่อแก้ปัญหาซอฟต์แวร์แบบทันท่วงที นอกจากนี้หากคุณสมัครแพ็กเกจแบบ Subscription (เช่าใช้รายเดือน) เราจะรับประกันฮาร์ดแวร์ให้ฟรีตลอดอายุสัญญา

ไม่พบคำตอบที่คุณตามหาใช่ไหม?
